## 🧠 Standard Troubleshooting Approach

Whenever an issue occurs, follow this structured approach:

### 1️⃣ Identify the Problem
- Understand exact error message  
- Capture logs/screenshots  

### 2️⃣ Understand the Flow
- Ariba → CIG → S4  
- Identify where failure occurred  

### 3️⃣ Validate Data
- Check master data  
- Verify document details (PR/PO/Invoice)  

### 4️⃣ Check Configuration
- Approval workflows  
- Mapping  
- Integration setup  

### 5️⃣ Analyze Logs
- CIG logs  
- System error messages  

### 6️⃣ Fix & Reprocess
- Correct issue  
- Reprocess transaction
