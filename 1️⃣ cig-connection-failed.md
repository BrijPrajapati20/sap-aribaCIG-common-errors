# ❌ Error: CIG Connection Failed

## 🧩 Problem
Ariba system is unable to connect with backend (S4/ERP) via CIG.

## 🌍 Scenario
During PO transmission, integration fails and document is not received in backend.

## 🔍 Root Cause
- Incorrect CIG configuration  
- Certificate issues  
- Endpoint mismatch  
- Network/firewall blockage  

## ⚠️ Common Errors
- Connection timeout  
- Endpoint not reachable  
- SSL/Certificate error  

## ✅ Solution
1. Check CIG connectivity status  
2. Validate certificates  
3. Verify endpoint URLs  
4. Check firewall/network  
5. Reprocess failed messages  

## 🧠 Approach
Start with connectivity → then configuration → then logs.

## 🎯 Learning
Most CIG issues are configuration or certificate related.
